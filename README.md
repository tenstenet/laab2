# Лабораторная работа 2. Виды операций и решение проблем узкого горлышка

## Тема
Многопоточность и асинхронное программирование на C#.

## Описание
В работе реализованы задания:

**Задание 1.1** — консольное приложение для подсчёта простых чисел от 1 до 10 000 в несколько потоков. Реализованы три версии синхронизации общего счётчика: Monitor (lock), Mutex и Semaphore. Выбор версии происходит через меню при запуске.

**Задание 1.2** — консольное приложение для обработки 15 наборов чисел в несколько потоков. Каждый поток считает сумму своего набора и записывает результат в общий лог. Используется Semaphore для ограничения числа потоков, Monitor (lock) для записи в лог, Mutex для подсчёта общей суммы.

**Задание 2** — два варианта консольного приложения для выполнения HTTP-запросов к трём серверам: синхронный и асинхронный.

## Файлы
- `Task1_1_Program.cs` — задание 1.1 
- `Task1_2_Program.cs` — задание 1.2 
- `data.txt` — 15 наборов по 100 случайных чисел для задания 1.2
- `Task2_Program.cs` — задание 2
